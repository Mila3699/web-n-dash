# Совместимость приложения

## Поддерживаемые устройства и браузеры

Приложение настроено для работы на современных устройствах за последние **6-7 лет** (с 2018 года).

### Минимальные версии браузеров:

- **Chrome**: версия 67+ (2018, поддержка BigInt)
- **Firefox**: версия 68+ (2019, поддержка BigInt)
- **Safari**: версия 14+ (2020, поддержка BigInt)
- **Edge**: версия 79+ (2020, на базе Chromium)
- **iOS Safari**: версия 14+ (2020)
- **Android Chrome**: версия 67+ (2018)

### Поддерживаемые операционные системы:

- **iOS**: 14 и выше (iPhone 6s и новее)
- **Android**: версия 6.0+ с Chrome 67+
- **Windows**: 7, 8, 10, 11
- **macOS**: 10.14 (Mojave) и выше
- **Linux**: Современные дистрибутивы

### Охват устройств

- **iPhone**: 6s и новее (2015+)
- **iPad**: 5th gen и новее (2017+)
- **Android**: Большинство устройств с 2018 года
- **Десктопы**: практически все за последние 7 лет

## Технические детали

### JavaScript
- Код транспилируется в **ES2020** стандарт
- Поддержка BigInt, async/await, Promise, spread operators
- Полная поддержка современных API
- Optional chaining и Nullish coalescing

### CSS
- CSS Grid и Flexbox
- CSS Custom Properties (переменные)
- Modern selectors и pseudo-classes
- Автоматические префиксы для старых браузеров

### Производительность
- Оптимизированная сборка для быстрой загрузки
- Code splitting для уменьшения начального размера
- Ленивая загрузка компонентов
- Оптимизация изображений

## Не поддерживается

- Internet Explorer 11 и ниже
- Opera Mini
- Очень старые браузеры (до 2018 года)
- Браузеры без поддержки BigInt

## Почему ES2020?

ES2020 выбран как минимальный target потому что:
1. Поддержка BigInt (используется некоторыми современными библиотеками)
2. Optional chaining и Nullish coalescing
3. Покрывает ~95% активных пользователей
4. Лучшая производительность и меньший размер бандла

## Тестирование

Для тестирования на различных устройствах используйте:
- Chrome DevTools (Device Mode)
- Firefox Responsive Design Mode
- Safari Web Inspector
- BrowserStack для реальных устройств
- Реальные устройства для финального тестирования

## Обновление конфигурации

Основные файлы конфигурации:
- `.browserslistrc` - список поддерживаемых браузеров
- `vite.config.ts` - настройки сборки Vite
- `index.html` - мета-теги для совместимости
- `tailwind.config.ts` - настройки Tailwind CSS
